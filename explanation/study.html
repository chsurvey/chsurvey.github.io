집현에서 공부할 내용

기본적인 웹 공부

  1. html을 활용한 기본적인 웹 페이지 공부

    2. 웹페이지를 작성하는 가장 기본적인 언어인 html을 '생활코딩' 유튜브를 활용하여 배운다.

    각 영상을 본 후 집현에서는 각자 만든 작업물을 공유하거나, 공동 제작으로 간단한 페이지를 만든다.

    매우 간단한 언어이기 때문에 빠르게 정리하고, 다음 단계로 나아간다.

  2. php를 활용한 웹 페이지 개선

    3. 웹페이지를 제작하는 또 다른 언어인 php에 대하여 공부

    html과 같이 직접 제작하고 공유하면서 빠르게 숙지하도록 한다.

Webhacking.kr 사이트 이용

webhacking이라는 사이트를 활용하여, 여태까지 배운 언어들을 가지고 웹해킹에 관한 문제 풀이

    기대되는 효과
  • 지금껏 배운 언어의 구조를 더 면밀하게 파악하고 활용 가능
  • 해킹이라는 분야에 간단하게 입문해보면서, 후에 네트워크 해킹과 보안을 활용한 해킹 대비

네트워크 해킹과 보안

네트워크 해킹과 보안 책을 활용하여, 기본적으로 컴퓨터의 구조와 네트워크의 작동원리에 관해 탐구
